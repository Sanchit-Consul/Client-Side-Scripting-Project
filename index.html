Equation of a Line Calculator

This is a calculator to calculate the equation of a line. The instructions on how to use this calculator are:

Using 2 Points:

  1. Enter the x-coordinate for the first point in the x1 input field.
  2. Enter the y-coordinate for the second point in in the y1 input field.
  3. Repeat the process for the x and y coordinates for the second point in the x2 and y2 input fields respectively.
  4. Click calculate. There should be a equation shown in the "y=mx+b" format with an illustration shown.
